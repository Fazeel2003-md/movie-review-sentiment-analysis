# Movies Sentiment Analysis
Overview

A simple movie review sentiment analysis app that predicts whether a review is Positive or Negative using an LSTM model.

Files

app.py – Streamlit app for user input and prediction.

tokenizer.pkl – Tokenizer used for preprocessing text.

sentiment.ipynb – Notebook used to train and test the model.

How to Run

Install required packages:

pip install streamlit keras numpy pandas

Run the app:

streamlit run app.py

Enter a movie review to see the sentiment prediction.
